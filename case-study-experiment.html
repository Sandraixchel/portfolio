Experiment Ops Toolkit

I built a lightweight experimentation hub that consolidates A/B test metrics, qualitative notes, and PM checklists inside a single Power BI experience plus automation scripts.

  • Power BI
  • SQL
  • Power Automate
  • A/B Testing
Timeline: Continuous · 2024-2025 Scope: 37 experiments · 5 product squads

Experiment Control Center

Filter by squad, experiment status, or hypothesis to drill into the uplift, guardrail metrics, and qualitative themes captured from live user sessions.

System design

Tests log directly into a SQL layer with governance around experiment metadata. Power Automate listens for new entries, syncs Confluence briefs, and posts Slack digests pointing to the BI hub.

Process wins

  • Unified backlog eliminated duplicate tests and aligned PMs around the same guardrails.
  • Automated weekly digests summarize KPIs and quotes so execs no longer wait for batch reviews.
  • Centralized qualitative tagging kept PMM and UX teams in sync with data scientists.

Future roadmap

I plan to bolt on a Bayesian calculator plus a Jira connector so experiment decisions sync automatically with delivery tickets.
